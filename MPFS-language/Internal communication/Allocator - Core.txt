У каждого ядра есть свой номер.
Исходя из номера ядра вычисляется ячейка с которой ядро читает данные. 
Формуля для вычисления ячейки> ячейка = Номер ядра * 6


0 + Ячейка | 1 
1 + Ячейка | 1 Прочитать
2 + Ячейка | [ ] Число с плавающей запятой [ 1 ] = да/[ 0 ] = нет.
3 + Ячейка | [ ] Число может быть отрицательным? [ 1 ] = да/[ 0 ] = нет.
4 + Ячейка | [ ] Сколько бит занимает число? (Если число с плавающей запятой то значения могут быть тоолько 16, 32, 64. Если число было введено не правильно ядро пишет 0 500)
5 + Ячейка | [ ] Номер бита с которого идёт чтение
_____________________________________________________________________________________________________
Ответы

0 + Ячейка | 200 (Всё нормально я закончил)
1 + Ячейка | [ ] Прочитаное число
2 + Ячейка | [ ] Бит после чтения
